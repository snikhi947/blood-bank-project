# 🩸 Blood Bank Management System

A full-stack Blood Bank Management System designed to connect blood donors, recipients, hospitals, and blood banks efficiently. This project helps users search for blood availability, register as donors, and manage blood requests digitally.

## 🚀 Features

- User Registration & Login
- Blood Donor Management
- Search Blood by Group
- Blood Request System
- Admin Dashboard
- Hospital/Blood Bank Management
- Donor Details Management
- Responsive User Interface
- Database Integration
- Authentication & Authorization
